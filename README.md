# skeleton
A Discord bot.

# To self-host:

## Prerequisites:
- `Node.js`  version 12 or higher
- `git` command line
- Discord bot token

## Steps:
1. Run `git clone https://github.com/DL302/skeleton.git`
2. Run `cd skeleton` and then run `npm install`
3. Rename `example_config.json` to `config.json` and put your bot token in

## To start up the bot:
Run `node index.js`
